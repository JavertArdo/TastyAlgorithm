# TastyAlgorithm
Simple implementation of the most popular numerical methods

## Already implemented

### Constant Value
* e
* PI

### Determine Max Point
* Dichotomy method
* Golden-Section method

### Determine Min Point
* Dichotomy method
* Golden-Section method

### Determine Zero Point
* Bisection method
* Secant method
* Newton-Raphson method
* Falsi method

### Function
* Derivative of function
* Integral of function
* Sinus
* Cosinus

### Extra
* Factorial
* Fibonacci numbers
* Power
